インストールと起動方法
＝＝＝＝＝＝＝＝＝＝＝

リポジトリーからコードを取得し、その下にvenv環境を用意します::

        $ git clone https://github.com/skgc45/pppt
        $ cd pppt
        # python3.6 -m venv venv
        # source venv/bin/activate
        (venv) $ pip install .
        (venv) $ norilog
         * Running on http://127.0.0.1:8000/
